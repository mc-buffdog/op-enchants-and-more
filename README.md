# OP Enchants and More
### ⚠️⚠️⚠️ Not production-tested: use at your own risk!!! ⚠️⚠️⚠️
---
### Adds new (overpowered) enchantments to the game
- Includes "Greater" versions of vanilla enchantments
- Not configurable (yet)
- **Obviously not Vanilla-friendly** (MIT License: you can fork this and make your own Vanilla+ version :D)

## Current enchantments
### Combat
| Enchantment | Description | On |
| --- | --- | --- |
| Cleave | Increased sweep attack area | Swords, Axes |
| Coup de Grâce | Chance to deal critical damage | Melee |
| Butterfly | Increased attack speed | Melee |
| Lifesteal | Heal on damage done[^1] | Melee |
| Greater Sharpness | Better version of _Sharpness_ | Melee |

[^1]: Lifesteal amount is based on target HP difference before and after attack. For example, killing a 10/20 HP mob by dealing 23 total damage with 10% lifesteal will only heal 1 HP.
